- Solc

- TestRPC (Dev)

- Public Ethereum Network (Production)

- Dapple (https://github.com/MakerDAO/dapple) - A tool for contract system developers.
- Embark (https://github.com/iurimatias/embark-framework) - Framework for Ethereum DApps
